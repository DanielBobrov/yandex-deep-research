# Yandex Deep Research (Deer Flow) - Backend Engine

Это система ИИ-агентов на базе [LangGraph](https://langchain-ai.github.io/langgraph/), предоставляющая мощный бэкенд, API (FastAPI) и возможности безопасного выполнения кода в песочнице (sandbox).

## 🚀 Особенности

- **Продвинутые ИИ-Агенты:** Использование LangGraph для создания сложных многошаговых цепочек рассуждений агентов.
- **Песочница (Sandbox):** Изолированное и безопасное окружение для выполнения сгенерированного кода, что позволяет агентам проводить исследования, запускать скрипты и анализировать данные без риска для основной системы.
- **REST API:** Шлюз (Gateway) на базе FastAPI для управления агентами, скиллами, MCP, памятью и потоками.
- **Современный стек:**
  - Python 3.12+
  - FastAPI, Uvicorn, LangGraph
- **Управление зависимостями:** Проект использует `uv` для быстрого и надежного управления пакетами Python.

## 📂 Структура проекта

- `backend/` — Серверная часть (FastAPI шлюз, агенты, логика LangGraph).
  - `backend/app/gateway/` — API роутеры (агенты, потоки, память, скиллы).
  - `backend/packages/harness/` — Ядро системы агентов (память, чекпоинты, песочница, MCP и др.).
- `scripts/` — Скрипты для запуска и обслуживания проекта.
- `docker/` — Конфигурация для запуска в контейнерах.
- `skills/` — Инструменты и скиллы для агентов.

## 🛠 Установка и запуск

### Требования
- Python >= 3.12
- Утилита [uv](https://github.com/astral-sh/uv)
- Docker (опционально, для запуска песочницы агентов)

### 1. Подготовка

Клонируйте репозиторий и создайте конфигурацию из примера:
```bash
make config
```

### 2. Запуск в режиме разработки

Скрипт `serve.sh` поднимет LangGraph сервер и Gateway API на локальной машине:
```bash
./scripts/serve.sh --dev
```

- **Gateway API:** `http://localhost:8000` (Документация: `http://localhost:8000/docs`)
- **LangGraph API:** `http://localhost:8123`

### 3. Производственный режим (Production)

Для запуска без hot-reload используйте:
```bash
./scripts/serve.sh --prod
```

## 📝 Разработка

Для форматирования и линтинга Python-кода в проекте используется `ruff`. 
Запуск тестов осуществляется с помощью `pytest`.

```bash
cd backend
uv run ruff check .
uv run pytest
```
