# Security Policy

## Supported Versions

As yandex-deep-research doesn't provide an official release yet, please use the latest version for the security updates.
Currently, we have two branches to maintain:
* main branch for yandex-deep-research 2.x
* main-1.x branch for yandex-deep-research 1.x 

## Reporting a Vulnerability

Please go to https://github.com/yandex/yandex-deep-research/security to report the vulnerability you find.
